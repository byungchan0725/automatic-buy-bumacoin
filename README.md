# coin-macro
부마코인 매수 매크로 프로그램 

<br>

# 사용 방법
## 0. 라이브러리 다운로드 
```python
pip install -r requirements.txt 
```


## 1. 사용 방법 
```python
python main.py
```

```
부마위키 아이디 입력: <자신의 아이디 입력>

부마위키 비밀번호 입력: <자신의 비밀번호 입력>

원하는 매수 가격 아래로: <ex 5만원 이하일 때 매수한다 -> 50000>

원하는 매도 가격 위로: <ex 200만원 이상일 때 매도한다 -> 2000000>
```
